Academic Eco Policy Evaluation Platform

MERN Stack Based Web Application

**Project Overview**

The Academic Eco Policy Evaluation Platform is a web-based system designed to help educational institutions digitally monitor, evaluate, and improve their eco-friendly policies. The platform enables tracking of energy usage, water consumption, waste management, plantation activities, and carbon footprint, and generates eco-performance scores through interactive dashboards.

This project is developed using the MERN Stack (MongoDB, Express.js, React.js, Node.js) to ensure scalability, performance, and real-world applicability.

**Features**

Secure Admin and Staff Login
Energy, Water, Waste & Plantation Data Management
Automatic Carbon Footprint Calculation
Eco Score Evaluation
Interactive Dashboards & Charts
Centralized Sustainability Reporting

**Tech Stack**
